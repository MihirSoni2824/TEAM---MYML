## 🎯 Overview

This project contains two main UI components:
1. **User Management Table** - A clean, professional user listing interface
2. **Blog Section** - A modern blog layout with cards and author information

Both components are built with responsive design principles and modern CSS techniques, featuring clean typography and intuitive user interfaces.

## 🧩 Components

### 1. User Management Table (by MIHIR)
- **File**: `HTML/MIHIR.html` + `CSS/MIHIR.css`
- **Description**: A comprehensive user management interface
- **Features**:
  - Sortable user table with name, title, email, and role columns
  - Add user functionality with prominent CTA button
  - Edit links for each user entry
  - Clean, minimalist design with professional styling
  - Responsive layout with proper spacing and typography

### 2. Blog Section (by Yash)
- **File**: `HTML/yash.html` + `CSS/yash.css`
- **Description**: Modern blog layout with featured articles
- **Features**:
  - Card-based blog post layout
  - Author information with profile pictures
  - Date and category tags
  - Responsive image handling
  - Clean typography with proper spacing
  - Professional color scheme

## 🚀 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/web-ui-components.git
   cd web-ui-components
   ```

2. **Open in browser**
   ```bash
   # For User Management Table
   open HTML/MIHIR.html
   
   # For Blog Section
   open HTML/yash.html
   ```

3. **Or use Live Server** (recommended)
   - Install Live Server extension in VS Code
   - Right-click on HTML files and select "Open with Live Server"

## 📖 Usage

### User Management Table
```html
<!-- Include the CSS file -->
<link rel="stylesheet" href="../CSS/MIHIR.css">

<!-- Use the HTML structure -->
<div class="main">
  <div class="header">
    <!-- Header content -->
  </div>
  <table>
    <!-- Table content -->
  </table>
</div>
```

### Blog Section
```html
<!-- Include the CSS file -->
<link rel="stylesheet" href="../CSS/yash.css">

<!-- Use the HTML structure -->
<div class="main">
  <h1>From the blog</h1>
  <div class="flex">
    <!-- Blog card content -->
  </div>
</div>
```
